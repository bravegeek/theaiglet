# The Aiglet

## The purpose of this project is to develop an AI agent to provide an easy way to have AI generate satirical articles.

The project aims to develop and document a structured approach for building and managing intelligent agents, focusing on clear planning, actionable tasks, effective workflows, and consistent prompting.

This project contains several key documents:

1. **planning.md**  
   Outlines the overall project goals, strategies, and high-level planning details.

2. **task.md**  
   Breaks down the project into actionable tasks, assigning responsibilities and tracking progress.

3. **workflow.md**  
   Describes the step-by-step workflow, including processes, tools, and best practices for collaboration.

4. **global_prompt.md**  
   Contains the main prompt or guiding instructions used by the agent throughout the project, ensuring consistency and clarity.
